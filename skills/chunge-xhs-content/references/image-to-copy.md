# Image To Copy

Use this guide when the user wants Xiaohongshu copy based on images, screenshots, covers, or image cards.

## Goal

Do not merely describe the image. Extract the likely posting intent behind the image, then write copy that helps publish it.

## Source types

### 1. Single photo or cover

Use when the user provides one main image.

- Infer the strongest mood, scene, and subject.
- Write a title that matches the visual hook.
- Keep the body light and readable.
- Use hashtags that reflect scene, style, and topic.

### 2. Multi-image card set

Use when the user provides a carousel or image-card series.

- Infer the main topic from repeated visual cues and big text.
- Avoid repeating the exact same card text in the body.
- Use the body to add transition, summary, takeaway, or call to save.
- If the cards are already educational, make the body more connective and digestible.

### 3. baoyu-image-cards output

Use when the images were generated by `baoyu-image-cards` or clearly follow that card logic.

- Read the image style, layout, and likely audience.
- Match the copy tone to the visual style.
- If the cards are dense, keep the body concise.
- If the cards are sparse and emotional, let the title and first line carry more hook.

## What to infer from images

Before writing, infer:

- topic
- intent
- audience
- emotional tone
- domain
- how much information is already visible on the image

## Writing rules

- The title should support the image, not compete with it.
- The body should add value beyond what is printed on the image.
- If the image already explains the content, use the body for framing and CTA.
- If the image is aesthetic-first, use the body to make the post feel more personal or useful.
- Keep output copy-safe and plain text as usual.

## Good patterns

### Visual-first lifestyle image

- title: mood or result first
- body: scene + feeling + why it is worth sharing

### Information card

- title: clear takeaway
- body: summary + who it helps + why to save

### Product collage

- title: use-case or benefit first
- body: who it fits + what stands out + practical recommendation

## Avoid

- describing every object in the image
- repeating all visible text line by line
- writing body copy that feels disconnected from the visual tone
