# Image Workflows

Use these workflows when `chunge-xhs-content` is paired with `baoyu-image-cards`.

## Goal

Make the final Xiaohongshu post feel like one coordinated package:

- images carry the visual hook
- copy provides framing, summary, and posting context
- title, body, and hashtags match the visual style and audience

## Workflow 1: Images already exist

Use when the user already has images or image cards.

1. Read the images first.
2. Infer topic, domain, audience, and emotional tone.
3. Check how much text is already visible on the images.
4. Write copy that complements the images instead of repeating them.

Best practice:

- dense cards -> lighter body copy
- sparse emotional visuals -> stronger title and first line
- product collage -> practical recommendation angle

## Workflow 2: User wants images and copy together

Use when the user wants a full Xiaohongshu post package.

1. Use `baoyu-image-cards` to determine visual strategy.
2. Observe or infer the chosen style, layout, and audience.
3. Write copy that matches that visual direction.
4. Keep the copy easy to paste and slightly broader than any one image panel.

Best practice:

- notion / knowledge-card style -> clearer takeaway and save-oriented body
- cute / lifestyle style -> warmer, more personal body
- bold / warning style -> stronger hook and more contrast in the title
- dense layouts -> fewer repeated details in the body

## Workflow 3: Copy first, then image alignment

Use when the user starts from a topic but wants the final copy to still feel image-aware.

1. Draft the main copy angle.
2. Anticipate what kind of image set would likely pair well with it.
3. Keep the title visually compatible with a cover card.
4. Keep the body from stealing all the information that should live on the image.

## Alignment checklist

Before finishing, check:

- does the title fit the image mood?
- does the body add value beyond the image text?
- do the hashtags match both the topic and the visual lane?
- does the final post feel like one coordinated package?

## Avoid

- repeating every card heading inside the body
- writing a dramatic title for calm minimalist visuals
- writing flat generic body copy for bold, high-energy visuals
- making both image and body equally dense
