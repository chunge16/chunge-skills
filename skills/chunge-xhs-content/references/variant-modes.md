# Variant Modes

Use these modes when the user asks for different strengths or posting styles.

## Modes

### 克制版

Use for professional, clean, low-noise copy.

- reduce emotional adjectives
- keep icons simple
- avoid dramatic contrast words
- emphasize clarity and usefulness

### 标准版

Use as the default mode.

- keep the tone natural and friendly
- balance hook, clarity, and rhythm
- make the copy feel native to Xiaohongshu without trying too hard

### 种草版

Use for recommendations and share-worthy notes.

- emphasize use case and concrete payoff
- add a little warmth and desire
- make the recommendation feel personal, not salesy

### 爆点版

Use for stronger stopping power.

- sharpen the first line
- increase rhythm and contrast
- keep the claims believable
- avoid fake urgency and low-trust hype

## Multiple variants

If the user asks for options, vary the angle and intensity together.
Do not generate nearly identical versions with minor word swaps.
