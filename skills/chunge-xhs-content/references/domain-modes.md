# Domain Modes

When the topic fits a recognizable Xiaohongshu content lane, adapt the copy so it sounds native to that lane instead of generic platform copy.

## How to use

1. Infer the closest domain from the user's material.
2. Adjust title rhythm, body emphasis, icon family, and hashtags.
3. Keep the base output contract unchanged:
   `标题` + `正文` + `话题`
4. If the domain is unclear, fall back to the default `标准` mode.

## Beauty / Skincare

Use for skincare, makeup, haircare, fragrance, beauty tools.

- Tone: soft, personal, experience-led
- Title focus: effect, texture, suitability, mood
- Body focus: who it suits, feel on use, standout benefit, caution if needed
- Icon direction: `✨ 💗 🌿 🫧`
- Hashtag style: product category + skin/hair type + scenario

Prefer:

- first-use feeling
- texture and finish
- skin-type fit
- real use scene

Avoid:

- hard claims that sound medical
- overpromising results

## Food / Cafe / Restaurant

Use for restaurants, cafes, recipes, snacks, drinks.

- Tone: lively, sensory, friendly
- Title focus: taste, atmosphere, value, whether it is worth trying
- Body focus: what to order, taste notes, price/value, scene recommendation
- Icon direction: `🍜 ☕️ 🍰 😋`
- Hashtag style: city + category + occasion

Prefer:

- concrete flavor words
- scene-based recommendations
- clear must-order suggestions

Avoid:

- vague praise with no dish details
- long storytelling before the recommendation

## Productivity / Tools / Apps

Use for SaaS, AI tools, workflows, apps, productivity systems.

- Tone: sharp, efficient, useful
- Title focus: result, time saved, workflow improvement
- Body focus: what problem it solves, who should use it, best feature, fastest use case
- Icon direction: `⚡️ 🧠 ✅ 📌`
- Hashtag style: tool category + workflow + target user

Prefer:

- outcome-first wording
- quick setup or quick win framing
- strong practical payoff

Avoid:

- vague innovation language
- feature dumping without user benefit

## Home / Lifestyle / Organization

Use for home items, cleaning, storage, room setup, daily-life improvements.

- Tone: cozy, practical, low-pressure
- Title focus: comfort, convenience, atmosphere, ease
- Body focus: what changed in daily life, where it works, why it is convenient
- Icon direction: `🪴 🏠 ☁️ 🧺`
- Hashtag style: home scene + item type + lifestyle intent

Prefer:

- before-and-after feeling
- small but real convenience
- everyday scenes

Avoid:

- overdescribing product specs
- sounding like a hard sell

## Fashion / Outfit / Accessories

Use for outfit posts, styling, bags, shoes, accessories.

- Tone: brisk, confident, aesthetic
- Title focus: styling effect, vibe, versatility, body/scene fit
- Body focus: how to wear it, what it matches with, why it flatters
- Icon direction: `🖤 👗 ✨ 👠`
- Hashtag style: item + outfit style + season/occasion

Prefer:

- silhouette and styling payoff
- clear matching ideas
- vibe words with restraint

Avoid:

- empty aesthetic adjectives with no outfit guidance
- too many trend buzzwords in one title

## Knowledge / Study / Self-Improvement

Use for educational posts, methods, summaries, book notes, learning tips.

- Tone: clear, structured, credible
- Title focus: takeaway, method, key point, mistake to avoid
- Body focus: conclusion first, 2-3 useful points, practical application
- Icon direction: `📚 ✅ 💡 📝`
- Hashtag style: topic + learning goal + method

Prefer:

- direct knowledge payoff
- clear structure
- one insight per line

Avoid:

- sounding like a lecture
- stuffing too many concepts into one post

## Fitness / Health / Wellness

Use for exercise, wellness routines, body management, healthy habits.

- Tone: encouraging, grounded, non-extreme
- Title focus: habit, comfort, consistency, realistic result
- Body focus: what changed, how to start, what to watch out for
- Icon direction: `🏃‍♀️ 🌿 💪 🫶`
- Hashtag style: routine + goal + scenario

Prefer:

- sustainable habit framing
- realistic expectations
- beginner-friendly language

Avoid:

- medical-style promises
- body shaming
- extreme transformation claims

## Parenting / Family

Use for parenting notes, family routines, child-related recommendations.

- Tone: warm, calm, experience-based
- Title focus: practical relief, family fit, ease of use
- Body focus: what problem it helps, what age or scenario it fits, real-life convenience
- Icon direction: `👶 🧸 💛 🏡`
- Hashtag style: parenting stage + scenario + category

Prefer:

- real family scene
- low-pressure advice
- practical reassurance

Avoid:

- absolute claims about children
- guilt-inducing tone

## Travel / City / Local Guide

Use for travel notes, city tips, itineraries, places to visit.

- Tone: inviting, scene-rich, compact
- Title focus: whether it is worth going, who it suits, strongest scene
- Body focus: what to expect, best time, route or planning tip, why it stands out
- Icon direction: `✈️ 📍 🌇 📷`
- Hashtag style: city + place type + travel intent

Prefer:

- location-specific usefulness
- visual scene cues
- easy planning details

Avoid:

- generic praise that could apply anywhere
- long diary-style setup before the useful part
