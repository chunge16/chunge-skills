# Body Patterns

Pick the pattern that matches the post intent.

## Seed note

Use for recommendation and sharing.

```text
正文
✨ 先说结论：一句话点明值不值得
💬 适合谁：说清适用人群或场景
📌 为什么喜欢：给出1-2个具体理由
🛒 怎么用/怎么买：给一个实用建议
```

## Tips note

Use for methods and experience.

```text
正文
✅ 先点题：这件事解决了什么问题
🧩 方法1：最关键的一步
📝 方法2：最容易执行的一步
🚀 方法3：最容易出效果的一步
```

## Warning note

Use for pitfalls and comparisons.

```text
正文
⚠️ 最容易出错的点
❌ 以前我怎么做错
✅ 后来我怎么改对
📍 给你一个最省事的判断标准
```

## Opinion note

Use for views and concise commentary.

```text
正文
💭 我的结论先放前面
📌 我为什么这么看
🔍 一个最关键的观察
🙋 你可以怎么判断适不适合自己
```
