# Title Formulas

Use one formula per draft. Keep the title compact and easy to understand.

## Formulas

- `icon + result`: `✨ 这样做真的省时间`
- `icon + contrast`: `🔥 以前踩坑 现在真香`
- `icon + checklist`: `✅ 新手一定要知道的3点`
- `icon + benefit`: `💡 这套方法太适合懒人了`
- `icon + warning`: `⚠️ 这个细节千万别忽略`
- `icon + emotion`: `🥹 原来舒服生活可以这么简单`

## Good habits

- Prefer one clear promise.
- Prefer daily words over formal words.
- Prefer short rhythm over stacked modifiers.
- Use numbers only when they genuinely help.

## Avoid

- Empty big words such as `颠覆`, `封神`, `天花板` unless the user explicitly wants strong hype.
- Too many icons.
- Titles that explain everything and leave no suspense.
