# Icon Guide

Use icons as visual anchors, not decoration noise.

## Default icon sets

- recommendation: `✨ 💖 🌿 🛍️`
- tutorial: `✅ 📝 🔧 🚀`
- warning: `⚠️ ❌ ✅ 📍`
- opinion: `💭 🔍 📌 🙋`
- daily life: `🌤️ ☕ 🫶 📷`

## Rules

- Use 1-2 icons in the title.
- Use one icon per body line.
- Keep the icon family consistent with the tone.
- Prefer simple, high-recognition icons.
- If the user wants a more professional tone, reduce decorative icons first.

## Avoid

- Mixing cute icons with serious warning topics.
- Using more than one icon at the start of every hashtag.
- Filling the body with symbols that distract from meaning.
