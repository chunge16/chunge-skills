# Hashtag Guide

Generate relevant, copyable hashtags for Xiaohongshu.

## Rules

- Default to 5-8 hashtags.
- Mix broad tags and specific tags.
- Keep tags readable and natural.
- Prefer nouns and common platform phrasing.
- Include one intent tag when helpful, such as `#好物分享` or `#避坑指南`.

## Construction order

1. Topic tag
2. Scenario tag
3. Audience tag
4. Intent tag
5. Optional trend or season tag

## Examples

- skincare note: `#护肤分享 #敏感肌护理 #平价好物 #护肤日常 #好物推荐`
- productivity note: `#效率提升 #自我管理 #时间管理 #实用技巧 #我的工作流`
- cafe lifestyle note: `#咖啡日常 #城市生活 #周末去处 #生活碎片 #氛围感`

## Avoid

- Repeating the same word with minor changes.
- Overly generic filler tags with no topic fit.
- Making up tags that users would never search.
