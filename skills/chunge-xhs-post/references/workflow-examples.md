# Workflow Examples

## Full package

```text
Use $chunge-xhs-post 帮我做一整套小红书图文内容，主题是“租房卧室改造”。先出图片方向，再生成可直接发布的标题、正文和话题。
```

```text
Use $chunge-xhs-post 做一套“小红书效率工具分享”图文内容。图片和文案都要，文案正文必须是纯文本，方便直接复制。
```

```text
Use $chunge-xhs-post 帮我做一个小红书内容，主题是“敏感肌换季护肤”。默认按整套图文输出，不要只给文案。
```

## Image-led

```text
Use $chunge-xhs-post 以图片卡片为主，帮我做一套“敏感肌换季护肤”的小红书内容，同时配一版不重复卡片文案的发布正文。
```

## Copy-led

```text
Use $chunge-xhs-post 围绕“Notion 管理工作流”做一套适合小红书发布的内容。重点先把文案写好，但要考虑后续配图风格和 baoyu-image-cards 卡片布局。
```

## Existing images

```text
Use $chunge-xhs-post 根据这组已有的小红书卡片图，补全一版适合直接发布的标题、正文和话题。不要重复图上的文字。
```

## Text-only

```text
Use $chunge-xhs-post 只输出文本，不生成图片。帮我做一套小红书发布文案，主题是“租房卧室改造”，要标题、正文、话题，正文必须是纯文本，方便直接复制。
```

```text
Use $chunge-xhs-post 这次不要图片，只要最终可发布文本。主题是“效率工具分享”，输出标题、正文、话题，不要解释，不要代码块。
```

## Image-only

```text
Use $chunge-xhs-post 这次只需要图片方向，不要文案。帮我规划一套“早八通勤穿搭”的小红书卡片视觉方案。
```
