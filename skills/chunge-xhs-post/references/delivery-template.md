# Delivery Template

Use this template when delivering a full Xiaohongshu package.

## Standard order

```text
图片
[image set, image direction, or image summary]

标题
[one-line title]

正文
[line 1]
[line 2]
[line 3]

话题
[#tag1 #tag2 #tag3]
```

## Notes

- If actual images were generated, summarize them briefly or present them directly.
- If only image direction is available, state the recommended style and layout briefly under `图片`.
- Keep `正文` copy-safe and plain text.
